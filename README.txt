INVITACIÓN PARA GORCHI

Subí esta carpeta completa o el ZIP a tu hosting.
El archivo principal es index.html y el video debe quedar al lado con el nombre giro-gorchi.mp4.

Estructura:
- index.html
- giro-gorchi.mp4

Para GitHub Pages:
1. Creá un repositorio.
2. Subí ambos archivos a la raíz.
3. Settings > Pages > Deploy from a branch > main / root.
